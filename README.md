Java Web Start CLI with OpenJDK8 for macOS
==========================================

javaws with OpenJDK8 for macOS 
(such as AdoptOpenJDK)

Build
-----
```
./build.sh
```



---
RUN

```
./bin/javaws [ARGS]
```
